Структуры возвращаемые в запросах
=====================================

Stock
---

Структура для описания бобины. 

|    Параметр   |                                                               Описание                                                               |
|:-------------:|--------------------------------------------------------------------------------------------------------------------------------------|
|    stockId    | идентификатор в системе                                                                                                              |
|     series    | серия                                                                                                 |
| start |  начальный номер                                                                                                                 |
|   current   | текущий номер                                                                                                             |
|  free  | остаток билетов в бобине                                                                                                              |
|    active    | признак активности бобины |
